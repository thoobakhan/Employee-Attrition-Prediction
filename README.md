# Employee Attrition Prediction

This repository contains the code used for the research paper:

**Evaluating Class-Weighting and SMOTE for Explainable Employee Attrition Prediction** 

## Dataset

IBM HR Analytics Employee Attrition and Performance dataset.

## Models

- Logistic Regression
- Random Forest
- XGBoost

## Class-Imbalance Settings

- Original dataset
- SMOTE
- Class weighting

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC

## Files

- `Employee_Attrition_Prediction.ipynb` — data preprocessing, model training, evaluation, feature importance, and confusion matrices.
